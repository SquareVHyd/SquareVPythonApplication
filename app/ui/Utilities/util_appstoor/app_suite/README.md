# Square V Suite Launcher

A beautiful, Windows 11 Fluent-inspired App Suite Launcher designed like a File Explorer. It allows you to organize and run your applications in one single interface.

## Applications Included
1. **Modern Calculator**: Standard arithmetic calculator with dark/light themes and full keyboard support.
2. **Business Base Calculator**: Financial helper to compute gross profit margins, markups, and cost vs. revenue details.
3. **Tax Calculator**: Simple income tax estimator displaying net income and tax owed.

## Features
- **File Explorer Layout**: Sidebar category tree structure, central item list view, and path navigation bar.
- **Dynamic Search**: Instant filtering of tools as you type in the search bar.
- **Integrated Windows**: Launched apps run as sub-windows (`tk.Toplevel`) of the main launcher process, creating a unified suite feel.
- **Branding**: Dedicated header and footer styling honoring "Square V Engineering Enterprises" and design by "govindayya.k".

## How to Run

1. Open a terminal in this directory.
2. Run the main launcher:
   ```bash
   python app_suite_launcher.py
   ```
